# Shubham Sunny — Website Modernization

Premium sales site for US business clients. Built with **Astro + React + Tailwind**, content driven from **Excel**.

## Quick start

```bash
npm install
npm run init:excel   # first time only — creates site-data.xlsx
npm run dev          # Excel auto-sync + dev server
```

## Project layout

| Path | Purpose |
|------|---------|
| `src/` | Pages, components, styles (edit here) |
| `public/` | Static assets, `robots.txt`, `llms.txt`, media |
| `site-data.xlsx` | Content source of truth |
| `api/` | Vercel serverless routes (`/api/chat`, `/api/visit`) |
| Root `index.html`, `_astro/`, etc. | **Published output** — auto-updated by CI on push; do not edit by hand |

## Update content

1. Edit `site-data.xlsx` at project root
2. Add images to `public/media/` (transformations, team, hero)
3. Save Excel — dev server auto-reloads

Or run manually: `npm run sync`

## Build & deploy

```bash
npm run build
```

Push to `master` triggers `.github/workflows/deploy.yml`: build → publish static files to repo root → GitHub Pages serves them.

## Email, chatbot & visit alerts (Vercel)

Deploy on **Vercel** (not GitHub Pages alone) so `/api/chat` and `/api/visit` work.

1. Copy `.env.example` → `.env` locally (never commit `.env`)
2. Add the same vars in Vercel → Settings → Environment Variables
3. Gmail: use an [App Password](https://myaccount.google.com/apppasswords) for `SMTP_PASS`

**Excel toggles** (Site sheet):

| field | values |
|-------|--------|
| `notify_on_visit` | `yes` / `no` — email when someone opens the site |
| `chatbot_enabled` | `yes` / `no` — floating chat widget |

## SEO & LLM discovery

- `/robots.txt` — search + AI crawlers (GPTBot, ClaudeBot, PerplexityBot, etc.)
- `/sitemap-index.xml` — auto-generated on build
- `/llms.txt` + `/llms-full.txt` — LLM-readable site summary (regenerated by `npm run sync`)
- JSON-LD structured data on every page (business, FAQ, reviews, services)
